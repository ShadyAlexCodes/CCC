CCC Library

The Code Central Curriculum (CCC) library provides helpful CSS stylings and easy to use JS functionality to help insturctors create new curriculum.

This library is incomplete!

It is expected that you understand how to edit html and css to get the results you need.
Use this library as one of many tools during development.

Please contribute to the Code Central Curriculum library as needed!
This is a great opportunity for you to improve your HTML/CSS/JS skills and contribute to an open source project.

Table of Contents

Setup

CCC.js runs a script that automatically creates a table of contents so you don't have to!

There are two html elements that CCC.js looks for.

  • tocHook
  • tocContent
CCC will 'hook' onto tocHook and start building a TOC there with the sections and subsections it finds in tocContent.

This setup is VERY important!
The JS and CSS are very dumb and will work if it is not setup exactly as it is below.

Copy text 
<!-- Header and imports above -->
<div class="contentWrapper">
  <div id="tocHook">CCC</div> <!-- tocHook w/ title "CCC" -->
  <div id="top" class="articalContent">
    <div id="tocContent"> <!-- tocContent w/ sections and subsection -->

      <!-- CONTENT -->
      
    </div>
  </div>
</div>
